A method of the markup without using the <nv-checkbox> on your HTML

You can use this custom checkboxes putting a keyword in the "is" attribute on element that specified by author.

The next checkboxes are the case of Overriding at the specified tags.

Paragraph Tag Checkbox

This checkbox was created using the "p" tag that's with the "is" attribute that was used "nv-p-checkbox" keyword.

DIV Tag Checkboxes

This checkbox was created using the "div" tag that's with the "is" attribute that was used "nv-div-checkbox" keyword.

Span Tag Checkboxes

This checkbox was created using the "span" tag that's with the "is" attribute that was used "nv-span-checkbox" keyword.

Anchor Tag Checkbox without href(the warning won't be shown on console) [Correct Usage]

This checkbox was created using the "a" tag that's with the "is" attribute that was used "nv-a-checkbox" keyword.

Anchor Tag Checkbox with href(the warning will be shown on console) [Incorrect Usage]

This checkbox was created using the "a" tag that's with the "is" attribute that was used "nv-a-checkbox" keyword.

The nv-a-tag checkbox is limited the using of the "href" attribute, because href tag give the focus.
